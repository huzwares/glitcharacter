# glitcharacter
kinda glitch effect
~~~~
<span id="glitchSpan">this is sample text</span>

<script>
var glitchTest = new glitch("glitchSpan", "random", "#aA!");
glitchTest.start();
</script>
~~~~

glitch(id, index of character to glitch or just "random", mask)


masks:
#: Numbers
a: Lower case
A: Upper case
!: Special characters
